KRISPI UI

Официальная документация библиотеки GUI-меню KRISPI UI для Roblox. Ниже описана установка, базовые компоненты и полный справочник по API.

Official documentation for the KRISPI UI GUI menu library for Roblox. Below you'll find installation instructions, core components, and a full API reference.

Стабильная версия Stable release Roblox · Luau

01 УстановкаInstallation

Подключите библиотеку одной строкой через loadstring, указав свою ссылку на raw-код.

Load the library in a single line via loadstring, pointing to your own raw source link.

luaulocal UI = loadstring(game:HttpGet("https://pastebin.com/raw/ТВОЙ_КОД"))()
Примечание. Замените ТВОЙ_КОД на идентификатор вашего raw-пастбина перед использованием. Note. Replace ТВОЙ_КОД with your own raw pastebin ID before use.

02 Создание окнаCreating a window

Метод CreateWindow создаёт главное окно интерфейса и принимает таблицу параметров.

The CreateWindow method creates the main interface window and accepts a parameters table.

luaulocal win = UI:CreateWindow("Название", {
    Width = 520,
    Height = 600,
    X = 0.5,
    Y = 0.5
})
ПараметрParameter ТипType ОписаниеDescription
WidthnumberШирина окна в пикселяхWindow width in pixels
HeightnumberВысота окна в пикселяхWindow height in pixels
X / Ynumber 0–1Относительная позиция на экранеRelative position on screen

03 ВкладкиTabs

Вкладки создаются внутри окна и служат контейнером для компонентов.

Tabs are created inside a window and act as containers for components.

luaulocal tab = win:CreateTab("Название вкладки")

КомпонентыComponents

▸ СекцияSection

luautab:CreateSection("Текст")

▸ КнопкаButton

luautab:CreateButton("Текст", function()
    print("Нажата!")
end)

▸ ТогглToggle

luautab:CreateToggle("Название", false, function(state)
    print(state)
end)

▸ СлайдерSlider

luautab:CreateSlider("Название", 16, 300, 16, "", function(v)
    print(v)
end)

▸ ДропдаунDropdown

luautab:CreateDropdown("Название", {"А", "Б"}, "А", function(v)
    print(v)
end)

Полный примерFull example

Ниже приведён минимальный рабочий пример, объединяющий окно, вкладку и все основные компоненты.

Below is a minimal working example combining a window, a tab, and all core components.

luaulocal UI = loadstring(game:HttpGet("https://pastebin.com/raw/ТВОЙ_КОД"))()
local win = UI:CreateWindow("Моё меню")
local main = win:CreateTab("Главная")
main:CreateSection("Управление")
main:CreateButton("Привет", function() print("Привет!") end)
main:CreateToggle("Ноклип", false, function(s) print(s) end)
main:CreateSlider("Скорость", 16, 300, 16, "", function(v)
    game.Players.LocalPlayer.Character.Humanoid.WalkSpeed = v
end)
main:CreateDropdown("Предмет", {"Бита", "Молоток"}, "Бита", print)

ЗакрытиеClosing

Полностью уничтожает интерфейс и освобождает связанные с ним ресурсы.

Fully destroys the interface and releases its associated resources.

luauwin:Close()
